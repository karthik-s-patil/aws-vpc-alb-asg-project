# aws-vpc-alb-asg-project

## Overview

This project demonstrates the implementation of a highly available and scalable web application infrastructure on AWS using a custom VPC, Application Load Balancer (ALB), and Auto Scaling Group (ASG).

The architecture follows AWS best practices by separating resources across multiple Availability Zones and ensuring high availability, fault tolerance, and scalability.

---

## Architecture Components

### Networking

- Custom VPC
- Public Subnets
- Private Subnets
- Internet Gateway
- NAT Gateway
- Route Tables

### Compute

- EC2 Instances
- Launch Template
- Auto Scaling Group

### Load Balancing

- Application Load Balancer (ALB)
- Target Groups
- Health Checks

---

## AWS Services Used

| Service | Purpose |
|----------|----------|
| VPC | Network Isolation |
| Subnets | Resource Segmentation |
| Internet Gateway | Internet Access |
| NAT Gateway | Outbound Access for Private Instances |
| EC2 | Compute Resources |
| ALB | Traffic Distribution |
| Auto Scaling Group | Automatic Scaling |
| Security Groups | Network Security |
